How to get the raw download link from GitHub

  1. Go to your GitHub repo and click the Excel File (e.g., Excel_Practice_Tasks.xlsx).
  2. Click the Raw button or open the file and then click Download.Copy the raw file URL (it begins with https://raw.githubusercontent.com/).
  3. Paste that URL into the GITHUB_RAW_URL location in the HTML above (and replace REPO_URL with your repo link).

Quick usage notes

  • Open the downloaded Excel file in Excel (or LibreOffice/ Google sheets.
  • Practice formulas: =AVERAGE(range), =SUM(range), =MAX(range), =MIN(range).
  • Create a chart (Insert-> chart) and try sorting/filtering the table.
